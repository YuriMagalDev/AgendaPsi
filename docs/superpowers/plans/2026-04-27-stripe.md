# Stripe Integration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Integrate Stripe for recurring subscription billing via three Edge Functions (checkout, webhook, portal). The PlanoPage "Assinar" buttons already call these functions (stub behavior from Plan 3 — now they become real).

**Architecture:** All Stripe logic lives in Edge Functions — no Stripe keys in the frontend. `stripe-checkout` creates a Checkout Session and returns the URL. `stripe-webhook` receives events from Stripe and updates `assinaturas`. `stripe-portal` creates a Billing Portal session for self-service management. Webhook is verified using `STRIPE_WEBHOOK_SECRET` signature before processing.

**Tech Stack:** Stripe SDK for Deno (`https://esm.sh/stripe@14?target=deno`), Supabase Edge Functions, `assinaturas` table (from Plan 2).

> **PREREQUISITE:** Plans 1, 2, and 3 must be applied first. `assinaturas` table and `PlanoPage` must exist.

## Task 3: Create stripe-checkout Edge Function

**Files:**
- Create: `supabase/functions/stripe-checkout/index.ts`

- [ ] **Step 1: Create the function**

```typescript
import { serve } from 'https://deno.land/std@0.177.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import Stripe from 'https://esm.sh/stripe@14?target=deno'

const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!, { apiVersion: '2024-04-10' })
const SUPABASE_URL = Deno.env.get('SUPABASE_URL')!
const SERVICE_ROLE_KEY = Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!

const PRICES: Record<string, string> = {
  basico:   Deno.env.get('STRIPE_PRICE_BASICO')!,
  completo: Deno.env.get('STRIPE_PRICE_COMPLETO')!,
}

const corsHeaders = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
  'Content-Type': 'application/json',
}

serve(async (req) => {
  if (req.method === 'OPTIONS') return new Response('ok', { headers: corsHeaders })

  const supabase = createClient(SUPABASE_URL, SERVICE_ROLE_KEY)

  // 1. Authenticate user
  const authHeader = req.headers.get('Authorization') ?? ''
  const token = authHeader.replace('Bearer ', '')
  const { data: { user }, error: userError } = await supabase.auth.getUser(token)
  if (userError || !user) {
    return new Response(JSON.stringify({ error: 'Não autenticado' }), { status: 401, headers: corsHeaders })
  }

  // 2. Get requested plan
  const body = await req.json() as { plano?: 'basico' | 'completo' }
  const plano = body.plano
  if (!plano || !PRICES[plano]) {
    return new Response(JSON.stringify({ error: 'Plano inválido' }), { status: 400, headers: corsHeaders })
  }
  const priceId = PRICES[plano]

  // 3. Get or create Stripe customer
  const { data: assinatura } = await supabase
    .from('assinaturas')
    .select('stripe_customer_id')
    .eq('user_id', user.id)
    .single()

  let customerId = assinatura?.stripe_customer_id
  if (!customerId) {
    const customer = await stripe.customers.create({
      email: user.email,
      metadata: { user_id: user.id },
    })
    customerId = customer.id
    await supabase
      .from('assinaturas')
      .update({ stripe_customer_id: customerId })
      .eq('user_id', user.id)
  }

  // 4. Create Checkout Session
  const origin = req.headers.get('origin') ?? 'https://app.agendapsi.com.br'
  const session = await stripe.checkout.sessions.create({
    customer: customerId,
    mode: 'subscription',
    line_items: [{ price: priceId, quantity: 1 }],
    success_url: `${origin}/plano?status=sucesso`,
    cancel_url: `${origin}/plano?status=cancelado`,
    metadata: { user_id: user.id, plano },
    payment_method_types: ['card'],
    locale: 'pt-BR',
  })

  return new Response(JSON.stringify({ url: session.url }), { headers: corsHeaders })
})
```

## Task 4: Create stripe-webhook Edge Function

**Files:**
- Create: `supabase/functions/stripe-webhook/index.ts`

- [ ] **Step 1: Create the function**

```typescript
import { serve } from 'https://deno.land/std@0.177.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import Stripe from 'https://esm.sh/stripe@14?target=deno'

const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!, { apiVersion: '2024-04-10' })
const WEBHOOK_SECRET = Deno.env.get('STRIPE_WEBHOOK_SECRET')!
const SUPABASE_URL = Deno.env.get('SUPABASE_URL')!
const SERVICE_ROLE_KEY = Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!

serve(async (req) => {
  const body = await req.text()
  const sig = req.headers.get('stripe-signature') ?? ''

  let event: Stripe.Event
  try {
    event = stripe.webhooks.constructEvent(body, sig, WEBHOOK_SECRET)
  } catch (e: unknown) {
    const message = e instanceof Error ? e.message : String(e)
    console.error('Stripe webhook signature verification failed:', message)
    return new Response(`Webhook Error: ${message}`, { status: 400 })
  }

  const supabase = createClient(SUPABASE_URL, SERVICE_ROLE_KEY)

  switch (event.type) {
    case 'checkout.session.completed': {
      const session = event.data.object as Stripe.Checkout.Session
      const userId = session.metadata?.user_id
      const plano  = session.metadata?.plano ?? 'basico'
      if (userId) {
        const { error } = await supabase
          .from('assinaturas')
          .update({
            plano,
            status: 'ativo',
            stripe_subscription_id: session.subscription as string,
            atualizado_em: new Date().toISOString(),
          })
          .eq('user_id', userId)
        if (error) console.error('checkout.session.completed update error:', JSON.stringify(error))
      }
      break
    }

    case 'invoice.paid': {
      const invoice = event.data.object as Stripe.Invoice
      const customerId = invoice.customer as string
      const { error } = await supabase
        .from('assinaturas')
        .update({
          status: 'ativo',
          atualizado_em: new Date().toISOString(),
        })
        .eq('stripe_customer_id', customerId)
      if (error) console.error('invoice.paid update error:', JSON.stringify(error))
      break
    }

    case 'invoice.payment_failed': {
      const invoice = event.data.object as Stripe.Invoice
      const customerId = invoice.customer as string
      const { error } = await supabase
        .from('assinaturas')
        .update({
          status: 'inadimplente',
          atualizado_em: new Date().toISOString(),
        })
        .eq('stripe_customer_id', customerId)
      if (error) console.error('invoice.payment_failed update error:', JSON.stringify(error))
      break
    }

    case 'customer.subscription.deleted': {
      const subscription = event.data.object as Stripe.Subscription
      const customerId = subscription.customer as string
      const { error } = await supabase
        .from('assinaturas')
        .update({
          status: 'cancelado',
          atualizado_em: new Date().toISOString(),
        })
        .eq('stripe_customer_id', customerId)
      if (error) console.error('customer.subscription.deleted update error:', JSON.stringify(error))
      break
    }

    default:
      console.log(`Unhandled event type: ${event.type}`)
  }

  return new Response(JSON.stringify({ received: true }), {
    status: 200,
    headers: { 'Content-Type': 'application/json' },
  })
})
```

## Task 5: Create stripe-portal Edge Function

**Files:**
- Create: `supabase/functions/stripe-portal/index.ts`

- [ ] **Step 1: Create the function**

```typescript
import { serve } from 'https://deno.land/std@0.177.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import Stripe from 'https://esm.sh/stripe@14?target=deno'

const stripe = new Stripe(Deno.env.get('STRIPE_SECRET_KEY')!, { apiVersion: '2024-04-10' })
const SUPABASE_URL = Deno.env.get('SUPABASE_URL')!
const SERVICE_ROLE_KEY = Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!

const corsHeaders = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
  'Content-Type': 'application/json',
}

serve(async (req) => {
  if (req.method === 'OPTIONS') return new Response('ok', { headers: corsHeaders })

  const supabase = createClient(SUPABASE_URL, SERVICE_ROLE_KEY)

  // 1. Authenticate user
  const authHeader = req.headers.get('Authorization') ?? ''
  const token = authHeader.replace('Bearer ', '')
  const { data: { user }, error: userError } = await supabase.auth.getUser(token)
  if (userError || !user) {
    return new Response(JSON.stringify({ error: 'Não autenticado' }), { status: 401, headers: corsHeaders })
  }

  // 2. Get stripe_customer_id
  const { data: assinatura, error: assinaturaError } = await supabase
    .from('assinaturas')
    .select('stripe_customer_id')
    .eq('user_id', user.id)
    .single()

  if (assinaturaError || !assinatura?.stripe_customer_id) {
    return new Response(
      JSON.stringify({ error: 'Nenhuma assinatura Stripe encontrada. Assine um plano primeiro.' }),
      { status: 400, headers: corsHeaders }
    )
  }

  // 3. Create Billing Portal session
  const origin = req.headers.get('origin') ?? 'https://app.agendapsi.com.br'
  const session = await stripe.billingPortal.sessions.create({
    customer: assinatura.stripe_customer_id,
    return_url: `${origin}/plano`,
  })

  return new Response(JSON.stringify({ url: session.url }), { headers: corsHeaders })
})
```
